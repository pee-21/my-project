# Quiz Project

A simple quiz application built with React.

## Features (planned)
- Multiple choice questions
- Score tracking
- Progress feedback
- User-friendly interface

## Setup
1. Clone this repo
2. Run `npm install`
3. Run `npm start` to launch the app
